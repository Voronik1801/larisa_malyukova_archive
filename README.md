## Как читать

Каждая статья — отдельный Markdown-файл с именем `YYYY-MM-DD — Заголовок.md`:

```
sources/
├── novayagazeta.ru/
│   ├── 2003/
│   │   ├── README.md                        ← индекс года
│   │   ├── 2003-01-09 — «ЗОЛОТОЙ ОВЕН» ХОЗЯИНА «КИНОТАВРА».md
│   │   ├── 2003-01-09 — ВЫСШИЙ ПИЛОТАЖ В АМУРНЫХ ДЕЛАХ.md
│   │   └── ...
│   ├── 2026/
│   │   ├── 2026-05-15 — И дым отечества.md
│   │   ├── images/                          ← фотографии статей
│   │   └── ...
└── kinoart.ru/
    └── <год>/<заголовок>.md
```

### Навигация
- **По годам** → откройте `sources/novayagazeta.ru/<год>/README.md` — там список со всеми статьями года.
- **По дате** → файлы внутри папок отсортированы хронологически.
- **Поиск** → используйте поиск GitHub по репозиторию или `grep -ri 'термин' sources/`.

![](images/<статья>/01.jpeg)   ← если есть
